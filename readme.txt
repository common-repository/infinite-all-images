== Description ==

= All the images of the media library to display in masonry and Infinite scroll. =
* You write and use short codes to page.
* Works with [Masonry](https://masonry.desandro.com/).
* Works with [Infinite Scroll](https://www.infinite-scroll.com/).
* [Demo](https://riverforest-wp.info/infinite-all-images-demo/)

== Installation ==

1. Upload `infinite-all-images` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add a new Page
4. Write a short code. The following text field. `[iai]`
